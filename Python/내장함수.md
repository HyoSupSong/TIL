## 내장함수
###내장함수는 별도의 모듈을 import 하지 않고 사용할 수 있는 함수를 말한다.

input()
> 사용자의 입력을 받는 함수
```python
# input은 스페이스도 문자열에 포함해서 받으므로 두 번 입력해줘야 한다.
# input은 문자열을 입력받기 때문에 숫자를 입력받는다면 int()로 바꿔줘야 한다.
name = input()
age = input()
name*3     # namenamename
int(age)*3 # 나이의 3배
```

lambda 매개변수 : 수행할 코드
> 익명함수를 만드는 함수
```python
# 11, 12, 13
print(list(map(lambda x : x+10, [1,2,3])))
# 30
print((lambda x, y : x+y)(10, 20))
```
